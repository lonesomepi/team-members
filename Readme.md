UFCLAS TEAM MEMBERS
===============

WordPress plugin that creates a custom post type for faculty and staff members

Requirements
------------
- WordPress

Installation
------------
- Install and activate for a site.

Page Templates
------------
- Individual Team Member
- All Team Members
  - Team Members are listed by category in alphabetical order
